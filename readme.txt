Pixigame - физдвижок на pixilang

1. Запуск:
1.1. Установите pixilang (https://warmplace.ru/soft/pixilang/)
1.2. Распакуйте архив с игрой (если вы читаете этот файл, значит, это уже выполнено)
1.3. Для этого пункта есть два способа:
1.3.1. Двойным щелчком ЛКМ запустите файл boot.pixi. Должно появиться окно с игрой.
1.3.2. Другой способ. Запустить pixilang, открыть в его файловом менеджере папку Pixigame из архива, запустить оттуда же boot.pixi. Появится окно с игрой.

2. Управление персонажем:
2.1. На компьютере:
2.1.1. [A], [D] => Ходить влево, вправо.
2.1.2. [Space] => Прыгать (если стоять на земле)
2.1.3. [Shift]/[C] => Накапливать рывок. При следующем нажатии клавиши движения рывок потратится, ускоря персонажа (только однократно).
2.2. На телефоне:
Нажатие на следующие части экрана будет воспроизводить следующее действие:
┌──────────────┬──────────────┐
│ Идти         │ Идти         │
│ влево        │ вправо       │3/4 от высоты экрана
│              │              │
├──────────────┼──────────────┤
│ Рывок +      │ Прыжок       │1/4 от высоты экрана
└──────────────┴──────────────┘
      1/2             1/2
 ширины экрана   ширины экрана

Также в обоих вырхних углах экрана выводятся сведения о состоянии: слева - динамические, справа - статические, т.е. константы. Константы можно задать свои (constants.txt). Удачи!

P.S. Не пытайтесь разбить персонажа о землю, не выйдет (но не факт).
